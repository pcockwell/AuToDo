Autodo

Automated scheduling API

Download as .zip Download as .tar.gz View on GitHub

Autodo is an in development scheduling API. This API will allow users to save tasks, fixed events, and preferences, and generate schedules that automatically take into account deadlines and indicate when each task should be worked on and completed.

Eventually this page will house the documentation for the API.

It is currently being worked on by:

  • Patrick Cockwell
  • Tony Zhang
  • Oscar Chow
  • Steven Miclette

It is currently being developed as a Fourth Year Design Project for the Software Engineering program at the University of Waterloo.
